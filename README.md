# Planta 
App en desarrollo

Esta app fue creada utilizando React js  + Vite y Firebase. Para poder ejectutarla de manera local el usuario debe comntar con su propia databease en Firebase.

## Correr de manera local

$ git clone 
$ cd 
$ npm install
$ npm run dev
